기능을 구현하기 전에 기능 목록을 만들고, 기능 단위로 커밋 하는 방식으로 진행
## <구현 할 기능 목록>

- [ ] 로또 구입 금액을 입력받는다.\
<예외처리 - [ERROR]로 시작>
    - [ ] 입력값이 숫자가 아닌 경우
    - [ ] 1,000원 단위가 아닌 경우
    - [ ] 입력값이 음수인 경우
---
- [ ] 로또 당첨 번호 6개를 ,로 구분하여 입력받는다.\
<예외처리 - [ERROR]로 시작>
    - [ ] 입력값이 숫자가 아닌 경우
    - [ ] 입력값이 1 ~ 45 범위가 아닌 경우
    - [ ] 입력값이 중복인 경우
    - [ ] 입력값이 6개가 아닌 경우
---
- [ ] 보너스 번호를 입력받는다.\
<예외처리 - [ERROR]로 시작>
    - [ ] 입력값이 숫자가 아닌 경우
    - [ ] 입력값이 1 ~ 45 범위가 아닌 경우
    - [ ] 입력값이 당첨 번호와 중복인 경우
---
- [ ] 구입 금액에 해당하는 로또 수량을 생성한다.
    - [ ] 발행한 로또 수량 및 번호를 출려한다.
    - [ ] 로또 번호는 오름차순으로 정렬한다.
    - [ ] Random값 추출은 pickUniqueNumbersInRange()을 사용한다.
--- 
- [ ] 구입 로또 번호와 당첨 번호, 보너스 번호를 비교하여 당첨 내역을 출력한다.
---
- [ ] 수익률을 계산하여 출력한다.
    - [ ] 수익률은 소수점 둘째 자리에서 반올림한다.
---
